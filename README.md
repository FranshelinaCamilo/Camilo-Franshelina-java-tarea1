## Tarea #1 - Menú interactivo

** Estudiante:** Franshelina Camilo Cabrera

** Matrícula:** 1000-5493

** Materia:** Programación 3

## Descripción:

Proyecto desarrollado en Java como una aplicación de consola que integra un menú interactivo con varias opciones funcionales. A través de este menú, el usuario puede ejecutar diferentes programas prácticos relacionados con temas básicos de programación vistos en los capítulos 1 al 5 del libro Java 2 Manual de Programación.

El sistema está diseñado para permitir el uso continuo de sus opciones, facilitando la navegación entre cada funcionalidad hasta que el usuario decida finalizar la ejecución.

## Características del proyecto: 

- Uso de menú interactivo con estructura switch
- Separación de funcionalidades en métodos independientes
- Aplicación de programación orientada a objetos
- Uso de clases, objetos, herencia y colecciones (ArrayList)
- Ejecución continua hasta la opción de salida
- Código estructurado y organizado para facilitar su lectura

## Tecnologías utilizadas: 
- Java
- Visual Studio Code
- JDK 25
